Overview

This software is a free MIDI file player.
It is designed to smoothly play very heavy MIDI files (commonly called "Black MIDI").

💡 Key Features: Lightweight and High Performance!

Launch & Uninstall

Run RainMIDI.exe to start the application.
To uninstall, simply delete the folder containing the executable.

Usage

RainMIDI requires no special configuration.
Just follow the steps below to play a MIDI file:

Basic Steps

  • Run RainMIDI.exe
  • Click the "Launch" button
  • Drag and drop a MIDI file into the window

FAQ

Q. No sound or strange sound.

Check the "MIDI Receiver" setting.
Default devices include "Gervill" or "Microsoft GS Wavetable SW Synth".
If not available, install a MIDI device.

Q. The app is laggy.

Turn ON "No use VRAM".
If still slow, set "Use RAM" to Low.
If still slow, enable "Invalidate Effect".
Otherwise, your PC may not meet requirements.

Q. App crashes during playback.

It may have exceeded available RAM.
Try launching "RainMIDI-Ram8G.exe" or "RainMIDI-Ram16G.exe".
